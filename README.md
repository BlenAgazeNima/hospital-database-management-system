## Project Overview

This project was developed as a final DBMS project and includes:

- Database design and normalization
- Entity Relationship Diagrams (ERD)
- Data Flow Diagrams (DFD)
- Oracle APEX implementation
- User interface and dialogue design
- System analysis and documentation
- Demo presentation and videos
